# dekranasda-bjn
Website Dewan Kerajinan Nasional Daerah Kabupaten Bojonegoro adalah platform informasi dan promosi kerajinan lokal yang dibangun menggunakan Laravel (PHP) dan MySQL. Fitur utama meliputi katalog produk kerajinan, profil pengrajin, agenda acara &amp; pelatihan, galeri, sistem berita, dan dashboard admin untuk manajemen konten dan statistik.
